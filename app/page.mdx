# Welcome to Nextra

This is your documentation homepage. Edit `app/page.mdx` to customize this content.

## Getting Started

1. Create MDX files in the `app` directory
2. The file structure will automatically generate the navigation
3. Customize the layout in `app/layout.jsx`

## Features

- **MDX Support**: Write markdown with React components
- **Auto-generated Navigation**: Pages are automatically discovered
- **Full-text Search**: Built-in search functionality
- **Theme Customization**: Customizable navbar, footer, and more
